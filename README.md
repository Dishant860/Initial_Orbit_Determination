# Angles-Only-IOD
Angles only initial orbit determination framework utilizing traditional and non-conventional methods

- Active Virtual Environments
- Install All Requirements from requirements.txt
- Go inside the folder name Angles-Only-IOD
- Run  Pyhton file angles_only_traditional_iod.py according to the Methods LAPLACE or GAUSS to get Output files of Methods
- Run Jupyter File EvaluationTraditional.ipynb to get Traditional Methods Results
- you will get all_parsed_data.csv file after running angles_only_traditional_iod.py from which you can verify that during evaulation it's taking the right data for MAE and RMSE Results
